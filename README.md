# Github Purple Theme

A dark VS Code theme inspired by GitHub's dark mode with purple accents.

## Description

Github Purple is a dark color theme for Visual Studio Code that combines GitHub's dark mode aesthetic with vibrant purple highlights. It is based on [jeff-j-chen/github-purple](https://github.com/jeff-j-chen/github-purple) with slight modifications for improved contrast and readability.

## Credits

- Based on [github-purple](https://github.com/jeff-j-chen/github-purple) by jeff-j-chen
- Modified for improved contrast and visual comfort
